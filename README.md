# Sistem-Monitoring-dan-Pengatur-Suhu-Pada-Kandang-Anak-Ayam-Berbasis-IoT-Blynk
Membuat sistem monitoring dan pengatur suhu pada kandang ayam yang dapat dikontrol secara manual dan otomatis berbasis IoT ESP32 sebagai kontroller. Kemudian komponen untuk mendeteksi suhu dan kelembapan menggunakan sensor DHT22, Blynk berfungsi sebagai monitor. Lalu untuk mengontrol suhu pada kandang anak ayam menggunakan Kipas 12v dan lampu pijar . Dimana jika suhu udara panas lebih dari 32째C dan suhu udara dinggin kurang dari 30째C, adanya pengontrol suhu pada kandang anak ayam suhu akan di normalkan menjadi 30째-32째C. Dengan begitu akan membuat suhu normal dan stabil yang membuat pertumbuhan ayam mejadi lebih baik.
